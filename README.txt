ABOUT
The game is Escape from AC.
It was asked by Academia de c√≥digo, the scope was to create a game or an application with a theme of our choice.
It's one escape game. It consists in one player that is trying to escape from the academy. In order to do it, he needs a key to unlock the doors that will lead him to the escaping gate. To obtain the key, he must answer correctly the questions of the master coders. 
The player has 3 beers (life). Every time he answer wrong a question, lose a beer. When he run out of beers, the game is over.

INSTRUCTIONS
The game has only 6 keys:
up, down, right, left (direction keys);
1, 2 and 3 (choose the answer).
